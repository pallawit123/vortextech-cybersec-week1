# Vortex Tech Cyber Security Internship – Week 1

## Researching the OWASP Top 10 Web Application Security Risks

This repository contains my Week 1 assignment for the Vortex Tech Cyber Security Internship.

The objective of this project was to research five vulnerabilities from the OWASP Top 10 (2021), understand how they work, analyze real-world incidents, and explore practical mitigation techniques.

---

## Objectives

- Study the OWASP Top 10
- Understand common web application vulnerabilities
- Learn how attackers exploit insecure applications
- Research real-world security incidents
- Explore mitigation techniques used by developers

---

## Vulnerabilities Covered

- Injection
- Broken Access Control
- Identification and Authentication Failures
- Cross-Site Scripting (XSS)
- Server-Side Request Forgery (SSRF)
